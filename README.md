# AuxiliumServices.Common
This project is a library of common code used by the various AuxiliumServices projects.
It should not be used on its own, but as an installable Nuget package.



## Installation
To install the AuxiliumServices.Common package, use the following command in the Package Manager Console:
```
Install-Package AuxiliumServices.Common
```
Or via the .NET CLI:
```
dotnet add package AuxiliumServices.Common
```
